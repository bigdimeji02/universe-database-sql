# 🌌 Universe Database

## Overview
I built this project to practice relational database design using PostgreSQL. The database models a simple universe with galaxies, stars, planets, and moons.

## What I Did
- Created tables for galaxies, stars, planets, and moons  
- Linked the tables using foreign keys  
- Inserted sample data  
- Wrote queries to explore the data  

## Queries
- Used JOINs to connect tables  
- Used WHERE to filter data  
- Used GROUP BY for aggregation  

## How to Run
1. Open PostgreSQL (psql)

2. Connect to a database:
   
   \c postgres

3. Run the SQL file:

   \i universe.sql

## What I Learned
- How to structure relational databases  
- How tables connect using keys  
- How to query across multiple tables
